jQuery: Javascript con jQuery

jQuery: es una libreria de javascript para trabajar con los objetos del DOM de un modo simplificado.

Cuando se utiliza una librería se debe adaptar a sus mecanismos de uso, el tratar de llevar los conocimientos que se tienen sobre un tema y aplicarlos a dicha librería puede ser complicado. Lo más adecuado es ir viendo cual es la mecánica de trabajar con dicha librería con problemas muy sencillos e ir complicándolo a medida que entendemos su funcionamiento.

Entre las ventajas que proporciona jQuery se pueden definir:

  • Ahorra lineas de código
  • Transparenta el soporte de las aplicaciones para los navegadores principales
  • Provee mecanismos para captura de eventos
  • Provee funciones para animar el contenido de la página de forma sencilla
  • Integra funcionalidades para trabajar con ajax
  • Existen un amplió abanico de extensiones

Si bien jQuery es una librería bastante antigua y actualmente hay otras tecnologías para implementar sitios web como Angular React y Vue, aun existen millones de sitios web que aun hacen uso de jQuery

Captura del evento click: Utilizando javascript tradicional y jQuery

Se dispondrá el problema trivial de capturar el evento de un control HTML de tipo button, el cual se realizará utilizando la metodología tradicional con javascript accediendo directamente a las funciones del DOM y luego empleando la librería jQuery:

Hacer clic sobre este párrafo para capturar el evento click con jQuery y enviar un mensake a consola web del navegador!! 

                            // Utilizando javascript clásico:
            
                            let iniciarEventos = function () {
                                let boton1 = document.querySelector('#javascript');
                                
                                boton1.addEventListener('click', e => {
                            
                                    alert('Evento "click" capturado con javascript clásico');
                            
                                }, false);
                            }
                        
                            // Utilizando jQuery:

                            $(document).ready( function() {

                                let boton = $('#jquery');
                                boton.click( () =>  {

                                    alert('Evento "click" capturado con jQuery');
                                });
                            });
