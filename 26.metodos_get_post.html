jQuery: Ajax: Métodos get( ) & post( )

Los métodos get( ) y post( ) de jQuery, se utilizan para solicitar datos al servidor mediante peticiones HTTP GET o POST.

El método GET se utiliza básicamente para conseguir (recuperar) algunos datos del servidor, este método puede devolver los datos almacenados en memoria caché

El método POST también se puede utilizar para obtener algunos datos del servidor, sin embargo, el método POST, nunca almacena datos en caché y con frecuencias es utilizado para enviar datos junto con la solicitud.

Nota: Utilizamos estas funciones cuando no queremos inyectar directamente los datos a nuestra página, sino que queremos procesarlos y eventualmente agregarlos en distintas partes.

Peticiones HTTP: Método get( ) vs post( )

Como se menciona en parrafos anteriores, estos 2 métodos son utilizados para realizar solicitudes/respuestas entre un cliente y el servidor.

  • Método GET: Pide los datos de un recurso especificado. Este método utiliza básicamente para conseguir algunos datos del servidor. El método get puede devolver los datos almacenados en caché
  • Método POST: Adjunta los datos para un recurso especifico, este método tambien es utilizado para solicitar datos, sin embargo, el método post no almacena datos en memoria caché, comunmente es utilizado para enviar datos en conjunto con la solicitud.

La funcion $.get es distinta a la ya utilizada $, en este caso es una función global de la librería jQuery:

Su sintaxis:

       
                    $.get(
                    URL,
                    data,
                    callback
                );

o bien:

       
                    jQuery.get(
                        URL,
                        data,
                        callback
                    );

La función $.post es idéntica a la función $.get, lo único que difiere es como el protocolo HTTP empaqueta los datos y los envía al servidor. Normalmente se utiliza la función $.post, ya que no está limitado la cantidad de datos que se pueden enviar al servidor.

Su sintaxis:

       
                    $.post(
                    URL,
                    data,
                    callback
                );

o bien:

       
                    jQuery.post(
                        URL,
                        data,
                        callback
                    );
