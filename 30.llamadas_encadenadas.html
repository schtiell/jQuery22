jQuery: Encadenamiento de Métodos

Otra característica muy utilizada en la librería jQuery es el encadenamiento de métodos, esto se puede hacer ya que casi todos los métodos de la clase jQuery retornan un objeto de tipo jQuery.

El encadenamiento de metodos es una técnica que permite ejecutar varios métodos de jQuery uno tras otro en el mismo elemento. Es decir se puede ejecutar múltiples métodos de jQuery sobre el mismo elemento dentro de un solo estado.

Por ejemplo:

            $("p").eq(1).css("background-color", "yellow");

Donde:

Selecciona el parrafo con indice 1 mediante el método eq( ) y le aplica estilos css con el método del mismo nombre

Patron chaining o Encadenamiento de métodos

Seguramente hemos visto que podemos hacer lo siguiente:

                            $('#main').addClass('MainSection').attr('role', 'main');

Se ejecuta la función $() e inmediatamente se ejecutan 2 o más métodos seguidos, sin necesidad de guardarlo en una variable. Todo ello en una línea. Así se evita hacer lo siguiente:

                            var main = $('#main');
                            main.addClass('MainSection');
                            main.attr('role', 'main');

Code...

                        let boton1 = $('#boton1');
                        boton1.click(function (e) { 
                            $('#descripcion')
                                .fadeTo('slow', 0.5, 'swing')
                                .hide('slow');
                        });

                        let boton2 = $('#boton2');
                        boton2.click(function (e){
                            $('#descripcion')
                                .show('slow')
                                .fadeTo("slow",1, "swing");
                        });

Primero se obtiene la referencia del id descripcion, la función $ retorna un objeto jQuery que se utiliza directamente para llamar al método fadeTo para aumentar su opacidad lentamente. Seguidamente el método fadeTo nos retorna la referencia del objeto jQuery y la utilizamos para llamar al método hide indicándole que oculte el div lentamente.
