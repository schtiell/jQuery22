jQuery: Método $.ajax( )

Hasta ahora se ha visto que jQuery provee varias formas de recuperar y enviar datos al servidor utilizando ajax a traves de jQuery

  • Método load( )
  • Funciones $.get( ) y $.post( )
  • $.getJSON( )

Existe otra función llamada $.ajax( ) que es la más completa de todas, pero como desventaja es más compleja su empleo. Recordando que AJAX es el arte de intercambiar datos con un servidor y actualizar partes de una página web, sin recargar toda la página.

Su sintaxis:

            $.ajax({
                name:value, 
                name:value, ... 
            });

El método ajax( ): se utiliza para realizar una solicitud AJAX (HTTP asíncrono). Todos los métodos jQuery AJAX usan el método ajax(). Este método se usa principalmente para solicitudes en las que no se pueden usar los otros métodos.

Ajax, retorna un objeto XMLHttpRequest que podemos eventualmente utilizarlo.
