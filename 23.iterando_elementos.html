jQuery: Iterando elementos con el método each()

jQuery dispone de un método que permite asociar una función que se ejecutará por cada elemento que contenga la lista del objeto jQuery.

Dentro de esta función se puede manipular el elemento actual

Su sintaxis:

            let x;
            x=$(elementos);
            x.each( function (){
                body function;
                ...;
            });
