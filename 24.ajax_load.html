jQuery: Ajax: Utilizando el método Load

El método load( ) es el más sencillo para recuperar datos del servidor.

La sintaxis:

            $(selector).load(URL,data,callback);

o bien:

            $(selector).load([nombre de la página],[datos],[funcion()])

Donde:

  • El parámetro de URL requerido especifica la URL que desea cargar.
  • El parámetro de data "opcional" especifica un conjunto de pares clave/valor de cadena de consulta para enviar junto con la solicitud.
  • El parámetro callback opcional es el nombre de una función que se ejecutará después de que load()se complete el método.
