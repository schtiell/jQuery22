jQuery: Evento blur

El evento blur se dispara cuando pierde el foco el control.

Sintaxis:

            $(selector).blur( function (){
                body function;
                ...;
            });

Se puede capturar el evento blur de un control de tipo text, textarea, button, checkbox, fileupload, password, radio, reset y submit.

Ejercicio:

Para probar su funcionamiento se dispone de dos controles de tipo text con algún contenido. Se Fija de color azul su fuente. Al tomar foco el control cambiará a color rojo y al perder el foco volverá a color azul.

Ejercicio: Confeccionar un formulario que contenga un text. Si al perder el foco su contenido es una cadena vacía mostrar mediante un alert tal situación.
