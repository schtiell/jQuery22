jQuery: Selectores de clases

Se debe recordar que se definen clases en CSS cuando una regla de estilo puede ser igual para un conjunto de marcas HTML.

Mediante este nombre de clase es posible acceder a todos los elementos utilizando la función $:

            let x = $(".className");
        
            let clases = (".className1 .className2")

Ejemplos
