jQuery: Métodos html( ) & html(valor)

El Método html( ), permite agregar un bloque de html a partir de un elemento de la página. Básicamente es la propiedad innerHTML del DOM. El método html() establece o devuelve el contenido (innerHTML de javascript) de los elementos seleccionados. Cuando se usa este método para devolver contenido, devuelve el contenido del primer elemento coincidente.

Cuando se utiliza este método para configurar el contenido, sobrescribe el contenido de TODOS los elementos coincidentes.

Sintaxis:

            $(selector).html(html content);

            o 

            $(selector).html(function (index, currentContent));

Por ejemplo:

            $("button").click(function(){
                $("p").html("Hello world!");
            });

El bloque de código jQuery muestra que la hacer click sobre el boton, inserta sobre el primer elemento parrado el bloque de código html indicado en el método html.

El método html() retorna el bloque html contenido a partir del elemento html que hace referencia el objeto jQuery

Ejercicio 1:

Problema: Disponer dos botones, al ser presionado el primero crear un formulario en forma dinámica que solicite el nombre de usuario y su clave. Si se presiona el segundo botón mostrar todos los elementos HTML del formulario previamente creado.

Ejericicio: Confeccionar una página que tenga dos botones. El primero debe mostrar en un alert el contenido HTML de la cabecera de la página y el segundo el contenido HTML del cuerpo de la página.
