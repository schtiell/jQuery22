jQuery: Evento mousemove

El evento mousemove se dispara cuando se mueve la flecha del mouse dentro del elemento HTML respectivo

Sintaxis:

            $(selector).mousemove(function (){
                body function;
            });

El evento mouseenter ocurre cuando el puntero del mouse está sobre (ingresa) el elemento seleccionado. El método mouseenter() activa el evento mouseenter o adjunta una función para que se ejecute cuando ocurre un evento mouseenter.

A diferencia del evento mouseover, el evento mouseenter solo se activa cuando el puntero del mouse ingresa al elemento seleccionado. El evento mouseover se activa si el puntero del mouse también ingresa a cualquier elemento secundario.

Sintaxis:

            $(selector).mouseenter(function (){
                body function;
            });

El evento mouseleave ocurre cuando el puntero del mouse deja el elemento seleccionado.A diferencia del evento mouseout , el evento mouseleave solo se activa cuando el puntero del mouse abandona los elementos seleccionados. El evento mouseout se activa si el puntero del mouse deja cualquier elemento secundario además del elemento seleccionado.

Sintaxis:

            $(selector).mouseleave(function (){
                body function;
            });

Si se quiere recuperar la coordenada donde se encuentra en ese momento el mouse, jQuery pasa el objeto event como parámetro (con la ventaja que hace transparente las diferencias entre IE y otros navegadores)
