jQuery: Uso del método getJSON

La función getJSON( ) hace una petición de datos al servidor considerando que retorna la información con notación JSON.

Su sintaxis es:

            $.getJSON(
                URL,
                Data,
                Callback(data,status,xhr)
            );

Donde:

  • URL: Es un parámetro obligatorio, se utiliza para especificar la URL del servidor al que se enviará la solicitud HTTP
  • Data: Es un parámetro opcional que especifica los datos que serán enviados al servidor.
  • Callback: Es un parámetro opcional que se ejecuta cuando la solicitud tiene exito.
