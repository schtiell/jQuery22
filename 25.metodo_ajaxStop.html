jQuery Método ajaxStop()

El método ajaxStop() se dispara cuando finaliza la petición de datos al servidor, es decir El método ajaxStop () especifica una función que se ejecutará cuando TODAS las solicitudes ajax se hayan completado.

Cuando se completa una solicitud ajax, jQuery verifica si hay más solicitudes AJAX. La función especificada con el método ajaxStop () se ejecutará si no hay otras solicitudes pendientes.

Sintaxis:

            $(document).ajaxStop(function (){
                body function;
                ...;
            })

Donde:

  • function ( ): La función dentro del método ajaxStop(), se ejecutará cuando se inicie la solicitud ajax

Nota: A partir de la versión 1.8 de jQuery, este método solo debe adjuntarse al documento.
