jQuery: Selector #id

La manera de declarar un bloque de código de jQuery, es la siguiente:

            $(document).ready(
                function (){
                    // jQuery methods go here..
                }

                function (){
                    // jQuery methods go here..
                }
            );

Otra forma de declarar un bloque de codigo jQuery de forma reducida es:

            $(function(){
                //jQuery methods go here
            });

Una vez explicado la forma de declaración de un bloque de jQuery, se pueden utilizar selectores de elementos. Para seleccionar un elemento específico se utiliza el atributo #id.

La identificación de un elemento por su id, debe ser único dentro de la página.

La sintaxis para seleccionar un elemento particular de la página mediante la propiedad id es:

$("#id")
