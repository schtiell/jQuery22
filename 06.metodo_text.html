jQuery: Método text(), text(valor)

Para saber el contenido de un elemento se utiliza el método text() establece o devuelve el contenido de texto de los elementos seleccionados.

Cuando se utiliza este método text() vacio sobre se obtiene el texto de todos los elementos coincidentes

Sintaxis

let x = $("#parrafo1");

luego si hacemos x.text() obtenemos el contenido del párrafo con id igual a parrafo1.

Luego si queremos cambiar el texto del párrafo deberíamos disponer:

Sintaxis

            let x = $("#parrafo1");
            x.text("Texto modificado...");

Pero hay que tener mucho cuidado cuando se utiliza jQuery ya que se puede cambiar el contenido de muchos elementos con una sola llamada al método text(), por ejemplo:

Sintaxis

            let x=$("p");
            x.text("Este texto aparece en todos los párrafos del documento");

El código anterior crea un objeto jQuery con la referencia a todos los párrafos contenidos en el documento. Luego si se llama al método text() enviándole una cadena, esta aparecerá en todo el documento remplazando el contenido de los párrafos actuales.
